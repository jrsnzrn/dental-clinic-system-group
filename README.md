# 🦷 BrightSmile Dental Clinic System

A modern **web-based dental clinic management system** built with React and Firebase.
Designed to help clinics manage patient records, appointments, and per-tooth treatment tracking efficiently.

## 📦 Installation (Local Development)

Clone the repository:

```bash
git clone https://github.com/jrsnzrn/dental-clinic-system.git
cd dental-clinic-system
```

Install dependencies:

```bash
npm install
```

Create your environment file:

```bash
cp .env.example .env
```

Fill in your Firebase config inside `.env`.

Run the development server:

```bash
npm run dev
```
